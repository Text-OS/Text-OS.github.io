Welcome to Text-OS

Welcome to the official website of Text-OS, a free, fully open source mini-OS made in batch. We're currently in developement stages, so if you want to help, send a Pull Request to our Github.
This website was designed by TEMPLATED with edits by RasmusOlle.

License
Text-OS uses the highly open license called "the MIT license". You can find the license at https://github.com/Text-OS/Text-OS/raw/master/LICENSE.txt

Download
Text-OS currently doesn't have a fixed build yet, however, you can download the latest source from https://github.com/Text-OS/Text-OS/archive/master.zip. Please stay tuned for when a fixed build arrives.

History

Text-OS Started in summer 2015 as a hobby, and has been developed by RasmusOlle and various other contributors from the open source community.
On nov 22 2015, I published Text-OS to github, and gives the public the first glance at Text-OS.
